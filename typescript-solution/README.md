# TypeScript solution

## Requirements

- Node.js 18+ (recommended: latest LTS)

## Setup

```bash
npm install
```

## Run tests

```bash
npm test
```

## Typecheck

```bash
npm run typecheck
```

## Build

```bash
npm run build
```

## Implementation

The function is implemented in `src/sort.ts` as:

- `sort(width, height, length, mass) -> "STANDARD" | "SPECIAL" | "REJECTED"`
